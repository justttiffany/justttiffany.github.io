Actors in a Graph Program

Created in March 2016

I created this Actors in a Graph program as part an assignment in my Advanced Data Structures course during my sophomore year in college. This assignment was completed in teams of two. Our task was to implement a program similar to that of the game, "Six Degress of Kevin Bacon," where we find the shortest amount of connections between actor Kevin Bacon and another actor/actress.

Task: Programmed an actor graph game in C++, based on the “Six Degrees of Kevin Bacon” game that finds the shortest path between actor Kevin Bacon and another movie actor in six steps or less.

Concepts Applied: Dijkstra's Algorithm, Maps, Queue

Technologies/Languages/Development Tools Used: C++, Vim
