Coursework Project

Auto-Complete Program

Basically like a Google Search
February 2016

Another way to Google Search

A program that mirrors the typical Google Search.

Overview

Google Search has made lives easier, providing limitless options for users of what they're looking for and simplifying the process of browsing. This program was created as part of an assignment for my Advanced Data Structures course during my sophomore year in college.

Skills

I developed this program in C++ and used the Vi/Vim text editor.

Task

The task was to program an Auto-Complete program that predicts the top-most occurring words of a given prefix based on user input. The application builds a dictionary of words and predicts the top-most occurring words in the dictionary of a given prefix based on user input.

Concepts

Because this was for a data structures class, many data structures were in use to implement this program. Some data structures used included the Hashtable, BST, Queue, and Multi-Way Trie. Breadth First Search was used as the search algorithm to search for the words.
