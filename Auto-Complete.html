Auto-Complete Program

Created in February 2016

I created this Auto-Complete program as part an assignment in my Advanced Data Structures course during my sophomore year in college. This assignment was completed in teams of two. Our task was to create an Auto-Complete program that mirrors a typical Google Search. The application builds a dictionary of words and predicts the top-most occurring words in the dictionary of a given prefix based on user input.

Task: Program an Auto-Complete program that predicts the top-most occurring words of a given prefix based on user input.

Concepts Applied: Hashtable, BST, Queue, Multi-Way Trie, Breadth-First Search

Technologies/Languages/Development Tools Used: C++, Vim
